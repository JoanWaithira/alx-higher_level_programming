=========================================
0-add_integer.py - Addition of Integers
=========================================
This module defines the function ``add_integer(a, b)`` which takes two parameters, 'a' and 'b', and returns their integer sum.

Usage
=====

Import the ``add_integer`` function:

	>>> add_integer = __import__('0-add_integer').add_integer

Valid Input Scenarios
=====================

1. Adding Two Integers:

You can add two integers together.
	>>> add_integer(1, 2)
   	3

   	>>> add_integer(2, 4)
   	6

   	>>> add_integer(-100, 30)
   	-70

2. Adding Integers with Float Inputs (Converted to Integers)
   	>>> add_integer(7.6, 2.7)
   	10

   	>>> add_integer(-2.3, 1.9)
   	-1

Invalid Input Scenarios
=======================

1. Adding Integer and Non-Integer (TypeError Expected)
